# Mini-server Node.js 

## Install 

    npm install

## Run the server

    npm run runServer

# REST API

The REST API to the example app is described below.

## List of requests for fiilms

> GET: /api/films
> GET: /api/films?id={param}

> POST: /api/films

> PUT: /api/films

> DELETE: /api/films

## List of requests for genres

> GET: /api/genres

> GET: /api/genres?id={param}

> POST: /api/genres

> PUT: /api/genres

> DELETE: /api/genres
